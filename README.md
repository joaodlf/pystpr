# PyStPr

**Python** **St**(atistics) & **Pr**(obabilities). Easy to use Data Analysis functions for developers.

Inspired from a number of sources,

* [Think Stats](http://greenteapress.com/thinkstats2/) by Allen B. Downey

### Installing
```
pip install pystpr
```
### Requirements:
* NumPy
* Pandas
* matplotlib
* Scipy
